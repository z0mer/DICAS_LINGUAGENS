# Tipos de Dados 

- Booleano (bool): `True / False`
- Inteiro (int): `14`
- Flutuante (float): `1.5`
- Palavra (string): `Python`
- Lista: `['Valor1','Valor2']`
- Dicionário: `{Chave1: Valor1, Chave2: Valor2}`
- Tupla: `('Valor1','Valor2')`

# Operadores Numéricos

- Adição: `+`
- Subtração: `-`
- Múltiplicação: `*`
- Divisão: `/`
- Exponenciação:`**`
- Resto: `%`
- Divisão Inteira: `//`

# Operadores Booleanos

- Lógica "e": `and`
- Lógica "ou": `or`
- Lógica "não": `not`
